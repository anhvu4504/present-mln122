# Present MLN122

Dự án presentation cho MLN122 được xây dựng bằng HTML, CSS và JavaScript.

## Mô tả

Dự án này là một website presentation đơn giản với giao diện hiện đại, responsive và các hiệu ứng tương tác.

## Tính năng

- ✅ Giao diện responsive, tương thích mobile
- ✅ Navigation menu với smooth scrolling
- ✅ Hero section với gradient background
- ✅ Card layout với hover effects
- ✅ Animation khi scroll vào view
- ✅ Modern UI/UX design

## Cấu trúc dự án

```
present-mln122/
├── index.html          # File HTML chính
├── css/
│   └── style.css      # File CSS styles
├── js/
│   └── script.js      # File JavaScript
├── README.md          # Tài liệu dự án
├── .gitignore         # Git ignore file
└── package.json       # Package configuration
```

## Sử dụng

1. Mở file `index.html` trong trình duyệt web
2. Hoặc sử dụng live server:
